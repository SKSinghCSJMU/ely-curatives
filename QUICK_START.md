# ⚡ QUICK START - 5 MINUTES SETUP

## Step 1️⃣: Install Node.js
Download and install from: https://nodejs.org/
(Choose LTS version)

---

## Step 2️⃣: Open Terminal/Command Prompt

**Windows:**
- Press `Win + R`
- Type `cmd`
- Press Enter

**Mac/Linux:**
- Open Terminal application

---

## Step 3️⃣: Navigate to Backend Folder

```
cd "d:\Web site\Ely Curatives\backend"
```

---

## Step 4️⃣: Install Dependencies

```
npm install
```

(This will download required packages - takes 2-3 minutes)

---

## Step 5️⃣: Start Server

```
npm start
```

You should see:
```
✓ ELY Curatives Backend running on http://localhost:3000
✓ Admin dashboard: http://localhost:3000/admin
```

---

## Step 6️⃣: Open Admin Dashboard

In your browser, go to:
```
http://localhost:3000/admin
```

---

## Step 7️⃣: Create Admin Account

1. Scroll down to "First Time? Register Admin"
2. Fill in:
   - Admin Name: `Your Name`
   - Email: `admin@elycuratives.com`
   - Password: `Your Password`
3. Click "Create Admin Account"

---

## Step 8️⃣: Login

Use the email and password from Step 7

---

## Step 9️⃣: Setup Payment Gateway

### For Razorpay (India):

1. Open dashboard
2. Go to "Payment Gateway" tab
3. Sign up at: https://razorpay.com
4. Copy API Key ID and Secret
5. Paste in admin dashboard
6. Click "Activate Razorpay"

---

## Step 🔟: Create Product List in Google Sheets

1. Go to: https://sheets.google.com
2. Create new spreadsheet
3. Add columns:
   ```
   Product Name | Category | Price | Description
   ```
4. Add your products
5. Share sheet (File → Publish to web)
6. Copy URL
7. In admin dashboard → E-Commerce tab
8. Paste URL
9. Click "Test Connection"

---

## ✅ You're Done!

Now:
- E-Commerce is ready
- Customers can buy from shop
- Orders appear in admin dashboard
- Payment works with Razorpay

---

## 🎯 To Enable E-Commerce on Shop

1. Login to admin dashboard
2. Go to "E-Commerce" tab
3. Toggle "E-Commerce Mode" ON
4. Save settings
5. Refresh `shop.html`
6. Products now show "Add to Cart" button!
